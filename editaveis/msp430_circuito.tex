\subsection{MSPGCC} % (fold)
\label{sub:mspgcc}

Is a port of the GNU C Compiler (GCC) and GNU Binutils (as, ld) for the embedded processor MSP430. Tools for debugging and download are provided (GDB, JTAG and BSL)
